GttRelease* gtt_get_latest_release_version ( cvector_vector_type(GttRelease *)  releases)

Gets the latest Release from the Vector of Releases.

Loops for each Release in the Vector, searches for MAJOR.MINOR.PATCH string in Release's version using /[0-9]+.[0-9]+.[0-9]+/ regexp, compares each number of the version and returns the latest Release.

Parameters
releasesVector of Releases in which to search for the latest Release.
Returns
The latest Release.
