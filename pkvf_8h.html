GTT_ERROR_EMITTER GTT_API cvector_vector_type ( GttPKVFToken  ) const

Parses PKVF to the form of token vector.

PKVF (Primitive Key-Value Format) is a simple internal data format. It is used while reading the information stored in GetterRelease files. Syntax:

key@#@value
another key@#@another value
array@#@element1@,@elemnt2

No quotes, no underscores in keys, only plain strings. New lines indicate new key-value pair. Key is separated from the value by the @#@ keys sequence. Note that neither key nor value cannot contain that sequence. Value can also be an array of strings - individual strings are divided by @,@ keys sequence.

Parameters
pkvfPKVF string. It must not be a string literal.
Returns
Vector with tokens read from PKVF.
