GTT_API void gtt_ptr_arr_delete_all ( GttPtrArr  self)

Frees each pointer in the array of pointers and deletes the array itself.

Do not call this function when at least one pointer in the array isn't heap-allocated. In such case gtt_ptr_arr_delete should be used instead.
