typedef GTT_HEADER_BEGIN struct GttScriptRunner GttScriptRunner

Representation of script runner.

Script runner is an object that is able to run the install/uninstall/upgrade script of the Release. The name of the script is usually GetterScript.<ext>, where <ext> determines the script runner that should be used to run that script.
